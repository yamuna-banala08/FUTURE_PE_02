Salon
📌 Project Overview
This project uses Prompt Engineering to generate high-converting UGC-style ad scripts for a local salon.
The goal is to create natural, relatable content that attracts customers and improves engagement on social media.

🏢 Business Selected
Sri Lakshmi Beauty Salon – Eluru
Target Audience: Women (18–40)
Platform: Instagram Reels / Ads

❗ Problem
Local salons often struggle with:

Low engagement on social media
Overly promotional ads
Lack of relatable content
Poor conversion rates
💡 Solution
This project builds an AI-based prompt system to generate:

UGC-style ad scripts
Multiple attention-grabbing hooks
Persuasive call-to-actions (CTAs)
Instagram captions
🧠 Prompt Framework
The system uses structured prompts to guide AI in generating content.

Key Structure: Hook → Problem → Experience → Result → CTA

🎯 Generated Content
Hooks
I found the best salon in Eluru 😍
If your hair looks like this… you NEED this
Girls in Eluru, you need to see this 👇
UGC Ad Scripts
Short, relatable scripts that mimic real customer experiences and highlight salon services like hair care and skincare.

CTAs
Book your appointment today!
Visit now and see the difference!
📱 Tools Used
ChatGPT (AI content generation)
Prompt Engineering (structured prompts)
📈 Outcome
Realistic, high-converting ad scripts
Improved engagement potential
Ready-to-use content for Instagram marketing
💰 Real-World Application
This system can be used for:

Freelancing (UGC script writing)
Social media marketing services
Local business promotion
🚀 Conclusion
This project demonstrates how AI and prompt engineering can help local businesses create authentic, engaging, and conversion-focused marketing content.
